# 李可老中医关于假流感防治的中医理论与实践方案

**来源**: https://d.biji.com/GIzBdwz1UurTdOqJ

## 核心防治原则

**寒者热之**：假流感属寒性疾病，应以温热方法防控；若已感染，需用信念破除病邪。

## 预防方案

### 一、预防方药组成

*   **基础方**：适用方加黄芩、苍穹、特兰、藿香、生晒参、乌龙梅、冰糖

*   **核心药材功效**：
    *   生黄芪：补大气、注重气、通血脉、补表气（建立内外屏障）
    *   信物：固护元阳
    *   藿香、长竹、佩兰：化食、化浊、化湿浊之气

### 二、特殊防护方法

*   **鼻腔防护**：苍穹和猫蜂黄打成粉，用三四年调成膏状，早晚涂抹鼻墙周围
*   **适用人群**：卫生部门提到的四种高危人群、孕妇均可使用

## 治疗方案

### 一、病性定位

*   **中医定性**：属太阳伤寒，小青龙汤证虚化
*   **核心病机**：本气先虚（人群普遍阳虚状态）

### 二、主方组成与用法（小青龙汤变通方）

| 药材 | 剂量 | 特殊处理 |
|------|------|----------|
| 麻黄 | 45克 | 先煎15分钟，出现浮沫后2分钟关火去沫 |
| 干姜 | 45克 | - |
| 五味子 | 33克 | - |
| 桂枝 | 45克 | - |
| 赤芍 | 45克 | - |
| 炙甘草 | 60克 | - |
| 栀子、紫丸、冬花 | 各45克 | - |
| 白果 | 20克 | 打碎搓泥30克 |

### 三、随症加减

*   **高热（38.5-39℃以上）**：加透明松石膏250克（需透明，灰色不可用）、明年36克
*   **咳喘、胸部不适**：加杏仁25克
*   **重症**：每剂加麝香0.2克（分三次服，首次全部服用）

### 四、用药注意事项

*   **麻黄减量原则**：首剂45克得汗后，后续减为5-10克
*   **附子选用**：需用泡素片（深复制经烘烤爆者），避免胆巴等炮制的附子
*   **煎药用水**：需用泡壶的螺丝水
*   **服药频率**：重症24小时内可连用三次

## 对当前防治方法的批判

*   **主流问题**：卫生部门采用温病派方法，使用大量寒凉药物清热解毒
*   **危害机制**：寒凉药物损伤人体阳气，撤掉防御屏障，导致"虚虚实实"
*   **临床后果**：增加危重症发生率和死亡率，与"寒者热之"原则南辕北辙

## 临床实践案例

*   **东莞塘厦镇案例**：弟子用原方三分之一剂量治疗小学爆发流感，全部治愈出院
*   **重症转化控制**：使用该方案可有效阻止轻症向重症转化，24小时内可见效
*   **特殊病例处理**：对使用芬达式打风出现精神异常的患者，改用小青龙汤后迅速恢复
