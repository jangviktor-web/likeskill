## 🎯 项目简介

**李可**（1930-2013），山西灵石人，52年临床实践，擅以重剂附子救治急危重症，自创方剂28首，累计治疗器质性心脏病约6000例，其中1000+例为西医放弃治疗的病危患者。

本skill将李可的辨证思维+彭子益圆运动理论完整转化为AI可执行的知识系统——从症状到辨证到方剂的全链路路由，10种假阳证+5种假阴证识别防止误诊。

## ✨ 核心特性

- **🏥 436个真实医案**：心衰61+中风10+肺病41+肿瘤51+其他273
- **🔍 15种假证识别**：假阳证10种（面红如妆/口腔溃疡/高热/咽痛等）+假阴证5种
- **💊 120+首经方剂量**：含李可临床剂量、彭子益圆运动解读、经方原方三版本
- **🧭 92种症状直通路由**：从症状直接到辨证到方剂的全链路
- **🧠 5个心智模型**：阳气为本/六经辨证/经方有效量/顾肾气保胃气/破格用药
- **🔄 六经辨证决策树**：四步辨证流程（表证→寒热虚实→兼证→假证识别）
- **📋 105+关键词索引**：精确到文件和医案编号
- **🛡️ 附子安全六要素**：炙甘草比例/黑小豆蜂蜜/煎煮时间/配干姜/逐步加量/瞑眩观察

### 一、六经辨证体系全面升级

#### 1.1 六经传变公式（新增）

```
太阳 → 阳明（表邪入里化热）
太阳 → 少阳（表邪进入半表半里）
太阳 → 少阴（直中少阴，肾元虚惫者多见）
太阴 → 少阴（太阴误治必传少阴）
少阴 → 厥阴（阴极阳生，寒热错杂）
```

#### 1.2 合病并病公式（新增）

| 合病/并病 | 方剂 |
|-----------|------|
| 太阳少阳合病 | 柴胡桂枝汤 |
| 太阳阳明合病 | 葛根汤/葛根芩连汤 |

| 少阴厥阴并病 | 四逆汤合乌梅丸 |

#### 1.3 辨证决策树（新增）

```
症状进来
├── 第一步：有无表证？
│   ├── 有汗脉浮缓 → 太阳中风（桂枝汤）
│   ├── 无汗脉浮紧 → 太阳伤寒（麻黄汤）
│   └── 无表证 → 第二步
├── 第二步：寒热虚实？
│   ├── 壮热大汗脉洪大 → 阳明（热实）
│   ├── 往来寒热口苦脉弦 → 少阳（半表半里）
│   ├── 腹满吐利食不下 → 太阴（虚寒）
│   ├── 脉微细但欲寐肢冷 → 少阴寒化
│   ├── 脉微细但口燥咽干腹满拒按 → 少阴热化·三急下（大承气汤）
│   ├── 脉微细心烦不得卧 → 少阴热化（黄连阿胶汤）
│   └── 厥热交替消渴气上撞心 → 厥阴（寒热错杂）
├── 第三步：有无兼证？
│   ├── 太阳+少阴 → 麻附细辛汤
│   ├── 太阳+阳明 → 大青龙汤
│   ├── 太阳+少阳 → 柴胡桂枝汤
│   ├── 少阳+阳明 → 大柴胡汤
│   ├── 三阳合病 → 白虎汤
│   └── 太阴+少阴 → 附子理中汤
└── 第四步：假证识别 🔴
    ├── 面红如妆+四肢冷 → 真寒假热（虚阳外越）
    ├── 脉沉实有力+舌红苔黄+肢冷 → 真热假寒
    ├── 无苔舌 → 可能是阳虚（李可独特见解）
    ├── 口腔溃疡/咽痛+清热无效+畏寒 → 龙火不藏
    └── 任何"上火"+寒凉药越吃越重 → 相火不在其位
```

### 四、安全机制强化

#### 4.1 附子安全公式（统一）

```
常规用药：炙甘草=附子量1/2~等量，文火久煎≥2h
急救场景：炙甘草≥附子量，武火急煎1h
```
