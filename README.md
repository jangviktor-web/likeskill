## 简介

**李可**（1930-2013），山西灵石人，52年临床实践，擅以重剂附子救治急危重症，自创方剂28首，累计治疗器质性心脏病约6000例，其中1000+例为西医放弃治疗的病危患者。被邓铁涛誉为"中医的脊梁"。

本skill提炼出李可完整的思维操作系统：

| 维度 | 数量 | 说明 |
|------|------|------|
| 心智模型 | 5个 | 阳气为本、六经辨证、经方有效量、顾肾气保胃气、破格用药 |
| 决策启发式 | 10条 | 生死关头救阳为急、诸症当先解表、病证冲突当从证等 |
| 医案 | 423个 | 心衰61+中风10+肺病41+肿瘤51+其他260 |
| 方剂 | 120+首 | 经方有效量总表+方剂索引 |
| 症状路由 | 92种 | 急症20+慢性病30+疑难杂症18+假证24 |
| 假证识别 | 24种 | 目赤/咽痛/高热/面红/无苔舌/糖尿病口渴/高血压面红等 |
| 关键词索引 | 105+条 | 常见疾病47+方剂31+理论27 |
| 经典语录 | 45条 | 38条已验证，7条待核实 |

## 快速开始

### 安装

```bash
# 克隆仓库
git clone https://github.com/jangviktor-web/likeskill.git

# 或直接复制到Claude Code skills目录
cp -r likeskill/ ~/.claude/skills/like-perspective/
```

### 使用方式

当用户提到以下场景时自动激活：

- 讨论急危重症的中医救治
- 询问附子、乌头等峻药的使用
- 讨论经方剂量问题
- 询问心衰、中风等急症的中医治疗
- 讨论扶阳理论、肿瘤治疗

### 回答流程

```
用户问题进来
├── 速查表有答案？→ 直接回答，不加载额外文件
├── 需要医案？→ cases/对应文件
├── 需要理论？→ modules/对应文件
├── 是假证？→ modules/05_zhenghou.md
└── 超出范围？→ 拒绝话术模板
```

### 2. 假证识别（24种）

> 「辨真假寒热是临证第一要务。误辨则误治，误治则杀人。」—— 李可

| 假证类型 | 症状 | 看似 | 实际 | 正确治法 |
|---------|------|------|------|---------|
| 真寒假热 | 高热+怕冷+脉沉细 | 实热证 | 阴盛格阳 | 四逆汤回阳 |
| 戴阳证 | 面红如妆 | 肝阳上亢 | 虚阳外越 | 四逆汤回阳 |
| 假火证 | 目赤/咽痛/口腔溃疡 | 实火 | 龙火不藏 | 引火汤+油桂 |
| 假阴虚 | 无苔舌 | 阴虚 | 阳虚甚至亡阳 | 四逆加人参汤 |
| 假消渴 | 糖尿病口渴 | 阴虚燥热 | 火不生土 | 附子理中汤 |
| 假肝阳 | 高血压面红头晕 | 肝阳上亢 | 浊阴窃踞阳位 | 温氏奔豚汤 |

## 已知限制

- 7条语录标注"待核实出处"（来自网页蒸馏，OCR源文件未收录）
- 部分医案存在OCR错误，已尽量修正
- 大剂量用药需在有经验的中医师指导下使用
- 不能替代李可本人的临床判断
- 李可逝世于2013年，对之后的疾病（如新冠）无直接经验
- 本skill仅供学习参考，临床处方须由执业中医师开立
