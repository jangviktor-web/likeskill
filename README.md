## 简介

李可老中医（1930-2013），山西灵石人，52年临床实践，擅以重剂附子救治急危重症，自创方剂28首，累计治疗器质性心脏病约6000例，其中1000+例为西医放弃治疗的病危患者。

本skill基于李可的13个来源（11本著作+2篇访谈），经过多轮蒸馏、验证、优化，提炼出完整的思维操作系统。

## 已知限制

- 7条语录标注"待核实出处"（来自网页蒸馏，OCR源文件未收录）
- 部分医案存在OCR错误，已尽量修正
- 大剂量用药需在有经验的中医师指导下使用
- 不能替代李可本人的临床判断
- 李可逝世于2013年，对之后的疾病（如新冠）无直接经验
- 本skill仅供学习参考，临床处方须由执业中医师开立
